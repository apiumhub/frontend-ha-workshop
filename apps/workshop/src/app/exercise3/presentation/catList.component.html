Cat entity is given with simple data for the purpose of this exercise. We need to implement 3 different repository implementations for getting the data needed to show in the table. Those repositories should be: using HttpClient from Angular, using Fetch and using Axios. (Axios is already installed, and there's a MockServiceWorker that will intercept the requests and return the data needed)
